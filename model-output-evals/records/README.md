# Eval Records

Add real model-output evaluation records here only after they have been
redacted and reviewed against `model-output-evals/model-output-eval-record.schema.json`.

Use stable filenames such as:

```text
2026-06-11-autopilot-control-plane-supervisor-routing-retry.json
```

Do not add raw prompts, raw model output, logs, transcripts, secrets,
credentials, customer data, or unredacted private context.
