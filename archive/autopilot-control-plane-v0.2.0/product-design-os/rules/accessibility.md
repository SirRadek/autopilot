# Accessibility Rules

Accessibility is part of Definition of Done, not a final polish step.

## Minimum Checks

- semantic headings
- keyboard navigation
- visible focus
- form labels
- contrast
- reduced-motion fallback
- responsive text fitting
- no text overlap
- no primary content hidden only inside canvas or video

Automated checks are useful but incomplete. Manual review remains required for
meaning, hierarchy, focus order, and motion comfort.
